## Описание продуктов

### SCHOOL PACK (349 ₽)
- 26 промптов для учебы в школе
- Все основные школьные предметы
- Структура сочинений и эссе
- Анализ стихотворений, решение задач

### STUDENT PACK (349 ₽)
- 30 промптов для студентов университетов
- Курсовые работы, дипломы, лабораторные
- Научные статьи, презентации для защиты

### PRESENTATION PACK (449 ₽)
- 20 промптов для создания презентаций
- Структура слайдов, дизайн-рекомендации
- Тексты для выступлений, работа с Canva/PPT

### SCIENCE PACK | STUDENT+ PACK (449 ₽)
- 30 специализированных промптов
- Форматы: IMRaД, PRISMA, NIH, COREQ, PICO, APA
- Для научных статей, дипломов, курсовых, исследований
- Методология, статистический анализ, академические навыки

## Технологии
- HTML5, CSS3, JavaScript
- Telegram WebApp API
- LocalStorage для сохранения покупок
- Пиксельный дизайн
